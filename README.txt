
-- SUMMARY --

TMGMT GlobalSight provides GlobalSight TMS integration for TMGMT module.


-- REQUIREMENTS --

Translation Management Tool (TMGMT)
  https://drupal.org/project/tmgmt

Nusoap library
  http://sourceforge.net/projects/nusoap/

GlobalSight instance
  Download source code: http://sourceforge.net/projects/globalsight/
  Have someone set it up for you: http://cmswithtms.net/


-- INSTALLATION --

Install as usual, see https://www.drupal.org/documentation/install/modules-themes/modules-8 for further information.
